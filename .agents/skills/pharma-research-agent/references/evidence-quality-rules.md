# 证据质量与结论规则

## 结论标签

| 标签 | 定义 | 使用条件 |
| --- | --- | --- |
| 直接证据 | 来源直接支持该事实或结论 | 有 PMID/DOI/NCT/openFDA 或文件页码支撑 |
| 综合判断 | 多个来源共同支持，需要分析归纳 | 至少两个来源或一个高质量来源加合理推理 |
| 待补充证据 | 当前检索不足以支持结论 | 数据缺失、来源冲突、仅有单一弱证据 |

## 文献证据排序

1. 系统综述、Meta 分析、指南或共识。
2. 随机对照临床研究。
3. 前瞻性队列或真实世界研究。
4. 回顾性研究。
5. 机制研究、动物实验、细胞实验。
6. 预印本、会议摘要、新闻稿。

## 数据源冲突处理

- 不覆盖冲突；并列表述冲突来源。
- 优先保留时间更新、字段更完整、ID 更稳定的数据。
- 对同一研究的论文和注册记录，注册记录用于试验设计，论文用于结果解释。
- 对 FAERS，只描述报告趋势和信号线索，不把报告数写成发生率或因果关系。

## 溯源检查清单

- 每个关键事实是否有来源 ID？
- 文献是否去重？
- 临床试验是否保留 NCT ID？
- openFDA 查询条件是否可复现？
- 报告是否区分直接证据、综合判断、待补充证据？
