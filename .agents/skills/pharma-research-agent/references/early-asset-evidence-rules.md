# 早期药物项目证据与结论规则

## 结论标签

| 标签 | 定义 | 使用条件 |
| --- | --- | --- |
| 支持证据 | 来源支持项目假设或可行性 | 有可追溯 ID，且与靶点、modality、模型或项目范围直接相关 |
| 反对证据 | 来源削弱项目假设或提示失败风险 | 同靶点失败、毒性、表达风险、模型不可转化、暴露不足、专利受限等 |
| 关键不确定性 | 当前证据不能判断，但会决定项目成败 | 数据缺失、来源冲突、只有弱证据或无法外推 |
| 综合判断 | 多维证据归纳后的第三方判断 | 明确列出支持证据、反对证据和不确定性后才能使用 |
| 待补充证据 | 检索不足或来源不可追溯 | 单一弱来源、公司材料未追溯、网页线索未验证 |

## 早期项目证据等级

| 等级 | 证据类型 | 解释 |
| --- | --- | --- |
| A | 人类遗传学、患者样本、多组学表达、临床 PoC、多个独立模型一致 | 对项目假设有强支持，仍需看适应症和 modality 是否匹配 |
| B | 体内动物药效、清晰 PK/PD、可解释 biomarker、同类项目成功 precedent | 支持继续推进，但需要转化和安全性去风险 |
| C | 体外机制、细胞药效、单一动物模型、结构或活性线索 | 只能支持早期假设，不能证明临床可行性 |
| D | 单一模型、过表达系统、公司材料、会议摘要、预印本、普通网页 | 弱证据；必须标注限制，不能单独支撑关键结论 |
| Negative | 同靶点失败、机制反证、正常组织高表达、毒性、暴露不足、模型不可转化 | 必须进入反对证据和失败路径分析 |

## 维度判断规则

### 靶点有效性

- 优先看人类遗传学、患者样本、疾病组织表达、功能扰动和临床相关性。
- 只有细胞或动物模型支持时，结论最多为“早期假设支持”。
- 靶点表达与疾病相关不等于疾病驱动；必须区分相关性和因果性。

### 制剂/技术路线

- 判断靶点定位是否适合 modality：细胞外/膜蛋白更适合抗体、ADC、双抗；细胞内靶点通常需要小分子、PROTAC、RNA 或细胞内递送方案。
- 递送、组织可及性、内吞、释放、半衰期、免疫原性和制造复杂度必须显式评估。
- 同 modality 成功 precedent 是支持证据；同 modality 系统性失败是反对证据。

### 非临床药效

- 记录模型是否过表达、是否使用疾病相关细胞/患者样本、是否有阳性/阴性对照、是否有剂量反应。
- 动物模型必须说明和人类疾病的相似点与不可比点。
- 无暴露-效应关系时，不得推断可行剂量。

### PK/PD 与安全

- PK/PD 判断必须连接剂量、暴露、靶点覆盖、PD biomarker 和药效。
- 正常组织高表达、物种差异和同类毒性必须进入安全风险。
- FAERS 或标签只能作为同类安全线索，不能直接证明早期项目风险发生率。

## 第三方结论要求

早期项目报告必须包含：

- `支持项目的证据`
- `反对项目的证据`
- `关键缺陷与失败路径`
- `建议补充实验`
- `可行性评分矩阵`
- `第三方结论`

## 可行性评分

| 维度 | 可选等级 | 判断依据 |
| --- | --- | --- |
| 靶点有效性 | 高/中/低/待补充 | 人类证据、机制、表达、模型一致性 |
| 制剂适配性 | 高/中/低/待补充 | 靶点定位、递送、同类 precedent、制造和安全约束 |
| 非临床转化可信度 | 高/中/低/待补充 | 模型相关性、剂量反应、PD biomarker、重复性 |
| PK/PD 可行性 | 高/中/低/待补充 | 暴露、靶点覆盖、组织分布、治疗窗 |
| 安全窗口 | 高/中/低/待补充 | 正常组织表达、毒理、同类风险、物种差异 |
| 差异化空间 | 高/中/低/待补充 | 竞品、失败先例、临床需求、机制差异 |
| 专利壁垒 | 高/中/低/待补充 | 核心专利、用途、组合、绕开空间 |
| 综合建议 | 推进/谨慎推进/暂缓/不建议 | 由支持证据、反对证据和关键缺口共同决定 |

## 溯源检查清单

- 每个关键判断是否有来源 ID？
- 是否同时列出支持证据和反对证据？
- 是否把体外或动物证据过度外推为临床可行？
- 是否说明模型、剂量、暴露、物种和样本限制？
- 是否明确最可能失败路径？
- 是否列出可执行的去风险实验或补充资料？
